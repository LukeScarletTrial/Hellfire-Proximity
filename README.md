# Hellfire Proximity

a Samsung DeX like experience for iOS. It allows the user to run Linux apps in a LENV (Linux Environment), you must be wondering why shouldn’t I just use a Linux machine with JIT, well Hellfire Proximity uses React Native and Objective-C to accomplish it’s goal the LENV only has to focus on the app/game only that means more performance for the app.

By the Way Hellfire Proximity is optimised for external disolay user.

# Minimal Requirements : 

 iPad 9th Generation or newer.
 iPhone 12 or Newer.

# Use Cases :

  if you want a laptop like experience on an external display,   download it right now!
